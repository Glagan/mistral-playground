# mistral-playground

## Get started

Install bun:

> https://bun.sh/docs/installation

And run the playground:

```bash
bun install
bun run dev --open
```

## Features

- Chat interface with the Mistral API
- All API options supported
- Customizable API endpoint to use your own
- History of your previous prompts
- Create Embeddings
